About Zupport

Zupport is a comprehensive system management tool that provides real-time monitoring and control of critical system components including:

  • Service status monitoring (Tomcat, Node-RED, Sprkz)
  • Queue monitoring with threshold alerts
  • System resource usage tracking
  • Disk space monitoring
  • Service management and restart capabilities
  • File editing and log viewing

Features

  • Real-time system status dashboard
  • Log file monitoring and filtering
  • Configuration file editing
  • Queue monitoring with file counts and age tracking
  • RESTful API for system management
  • Dark/Light theme support
